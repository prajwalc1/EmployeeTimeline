Schwarzenberg Tech Zeiterfassung - Installation Guide
==================================================

Thank you for choosing Schwarzenberg Tech's Time and Leave Management System. 
Follow these steps to install and set up the application:

1. System Requirements
   - Windows 10 or newer (64-bit)
   - At least 4GB RAM
   - 500MB free disk space
   - Administrator rights for installation
   - Active internet connection for initial setup

2. Installation Steps
   a. Extract the installation package to a preferred location
   b. Right-click on "install-windows.bat" and select "Run as administrator"
   c. The installer will:
      - Create necessary application directories
      - Set up the database
      - Create desktop and start menu shortcuts
      - Launch the application automatically
   d. If prompted by Windows Security, click "Yes" to allow the installation

3. First Start Guide
   a. Initial Setup:
      - The system will automatically initialize the database
      - You'll be prompted to create an admin account
   b. Default test credentials (for evaluation only):
      Username: test@example.com
      Password: test123
   c. Important: Change the default password immediately after first login

4. Features
   - Employee time tracking with German work time regulations
   - Leave management and approval workflow
   - Reporting and analytics
   - Data export (Excel, PDF)
   - Multi-user support with role-based access
   - Automatic break time calculations
   - Holiday calendar integration

5. Troubleshooting
   If you encounter any issues:
   a. Verify system requirements are met
   b. Ensure Windows is up to date
   c. Common solutions:
      - If the app won't start: Check Windows Event Viewer for details
      - Database errors: Verify write permissions in %APPDATA%\SchwarzenbergTech
      - Login issues: Clear application data and restart
   d. For technical support:
      - Email: support@schwarzenbergtech.com
      - Phone: +49 (0) XXX XXXXXXX
      - Hours: Mon-Fri, 9:00-17:00 CET

6. Updates
   - The application checks for updates automatically
   - Updates can be installed without losing your data
   - Manual updates can be triggered from the Help menu
   - Update history is available in the About section

7. Data Security
   - All data is stored locally in your %APPDATA% directory
   - Automatic daily backups are created
   - Data is encrypted at rest using industry-standard encryption
   - Compliance with GDPR and German data protection laws

8. Uninstallation
   - Use Windows Settings > Apps to uninstall
   - Your data will be preserved unless you choose to remove it
   - Backup your data before uninstalling if needed
